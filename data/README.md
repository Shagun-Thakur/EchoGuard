# Dataset

This project uses the MIMII Dataset.

The dataset is not included in this repository due to its size and licensing.

Current experiments use:

Machine Type:
- Pump

Machine ID:
- id_00

Channel:
- 0

After downloading the dataset, place it in the following structure:

```
data/
└── MIMII/
    └── pump/
        └── id_00/
            ├── normal/
            └── abnormal/
```
